# Flood

Simple ping flood tool

```sh
$ flood --help 
Ping flood tool

Usage: flood [OPTIONS] <HOST>

Arguments:
  <HOST>  Destination address

Options:
  -s, --size <SIZE>        Packet size in bytes [default: 1471]
  -t, --threads <THREADS>  Number of threads [default: 3]
  -h, --help               Print help
  -V, --version            Print version
```
